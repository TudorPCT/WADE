## 	:toolbox: Getting Started

<!-- Prerequisites -->
### :bangbang: Prerequisites

- Python 3.12

- Node.js

- npm

- PostgreSQL

- Apache Jena Fuseki


<!-- Installation -->
### :gear: Installation

Clone the repository:

```bash
  git clone https://github.com/TudorPCT/WADE.git
  cd WADE
```

Install dependencies:
```bash 
  # Frontend
  cd frontend
  npm install

  # Backend
  cd ../backend
  pip install -r requirements.txt
```
<!-- Run Locally -->
### :running: Run Locally

Start the backend:
```bash
  cd backend-service
  python app.py
```

Start the frontend:
```bash
  cd frontend
  npm start
```
